the structure of the repo of our team is:
```text
your-repo/

+-- .gitignore
+-- code/

    +-- README.md (a brief introduction to naive gdocs)
+-- docs/

    +-- DesignReport.md  (how we design the system)
    +-- TestingReport.md (how we test the system)
    +-- Lab2-gDocs.pdf   (The lab description)
```
